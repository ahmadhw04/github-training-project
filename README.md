# Zain Hotspot Portal

## Project Description

A hotspot login portal designed for internet service providers. The portal provides users with login functionality, service information, pricing plans, and hotspot status tracking.

## Features

* User login system
* Pricing section
* Sales points information
* User status monitoring
* Responsive interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

* css/
* js/
* img/
* fonts/
* HTML Pages

## Installation / Run Steps

1. Download the project files
2. Open index.html in your browser
3. Use the hotspot portal interface
